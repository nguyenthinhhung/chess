Lichess API Token

Required for importing games to your Lichess account.

Create one at lichess.org/account/oauth/token/create — no scopes need to be selected, just give it a name and submit.

How to use:
  1. Save your token above.
  2. Open any chess.com game page — an Analyze on Lichess button appears.
  3. Or click the extension icon to batch-import your last N games.
